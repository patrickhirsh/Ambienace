AMBIENCE NODE
Webserver API



/GetColor?Color=<uint8_t>
Gets the color (HSVW value) of the color indexed at 'Color'.

Returns:
{'Color': {'H':<uint16_t>, 'S':<uint8_t>, 'V':<uint8_t>, 'W':<uint8_t>} }


/SetColor?Color=<uint8_t>&H=<uint16_t>&S=<uint8_t>&V=<uint8_t>&W=<uint8_t>
Sets the color (HSVW or HSV value) of the color indexed at 'Color' with 'H', 'S', 'V', and optional 'W' values.

Returns:
None


/GetMode
Gets the mode the LEDs are currently in.

Returns:
{'Mode':<string>}


/GetModes
Gets all LED modes available for this device.

Returns:
{'Modes':[<string>]}


/SetMode?Mode=<string>
Sets the LED mode. Expects a valid string 'Mode' name.

Returns:
None


/GetBrightness
Gets the current brightness of the LEDs

Returns:
{'Brightness':0..255}


/SetBrightness?Brightness=<uint8_t>
Sets the brightness of the LEDs

Returns:
None


/GetActive
Gets the active (power) state of the LEDs

Returns:
{'Active':<bool>}


/SetActive?Active=<bool>
Sets the active (power) state of the LEDs

Returns:
None


/GetState
Gets all the current state information of the device in a single request

Returns:
{
'Color1': {'H':<uint16_t>,'S':<uint8_t>,'V':<uint8_t>,'W':<uint8_t>},
'Color2': {'H':<uint16_t>,'S':<uint8_t>,'V':<uint8_t>,'W':<uint8_t>},
'Color3': {'H':<uint16_t>,'S':<uint8_t>,'V':<uint8_t>,'W':<uint8_t>},
'Mode':<string>,
'Modes':[<string>]
'Brightness':<uint8_t>,
'Active':<bool>
}


/GetStats
Gets the current TickTime (ms), Average TickTime (ms) and Average TickRate (ticks per second)

Returns:
{'Current TickTime':<uint8_t>, 'Average TickTime':<float>, 'Average TickRate':<float>}
