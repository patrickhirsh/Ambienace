Ambience Node

API Reference:

/SetColor

- sets the color (HSVW or HSV value) of the color indexed at 'Color' - Color=<1..3> - H=<0..65535> - S=<0..255> - V=<0..255> - W=<0..255> (Optional) RETURNS: none /GetColor - gets the color (HSVW value) of the color indexed at 'Color' - Color=<1..3> RETURNS: {'Color': {'H':<0..65535>,'S':<0..255>,'V':<0..255>,'W':<0.255>} } /SetMode - sets the mode. Expects a valid string 'Mode' name - Mode= RETURNS: none /GetMode - gets the current mode RETURNS: {'Mode':} /SetBrightness - sets the brightness - Brightness=<0..255> RETURNS: none /GetBrightness - gets the current brightness RETURNS: {'Brightness':<0..255>} /SetActive - sets the active (power) state of the leds - Active= RETURNS: none /GetActive - gets the active (power) state of the leds RETURNS: {'Active':} /GetState - gets all the current state information of the device in a single request RETURNS: { 'Color1': {'H':<0..65535>,'S':<0..255>,'V':<0..255>,'W':<0.255>}, 'Color2': {'H':<0..65535>,'S':<0..255>,'V':<0..255>,'W':<0.255>}, 'Color3': {'H':<0..65535>,'S':<0..255>,'V':<0..255>,'W':<0.255>}, 'Mode':, 'Brightness':<0..255>, 'Active': } /GetStats - gets the current performance metrics of the leds {'Current TickTime':, 'Average TickTime':, 'Average TickRate':}
